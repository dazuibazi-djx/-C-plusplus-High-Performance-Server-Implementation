#本项目是一个仿 muduo 库的 One Thread One Loop 式主从 Reactor 模型高并发服务器组件 。主要的核心代码为server.hpp部分，有test测试代码、http简单的服务器代码。
获取的新连接会分发给子 Reactor 进行通信事件监控、数据读写及业务处理 。
项目目标：通过该组件，可以简洁快速地搭建高性能服务器 。
业务解耦：项目定位为基础组件，本身并不包含实际的业务内容 。
##核心架构：主从 Reactor 模型
主 Reactor：主线程仅仅负责监控监听描述符以获取新建连接，这保证了获取新连接的高效性，从而提高了服务器的并发性能 。
子 Reactor：主 Reactor 获取到新连接后，会将其分发给子 Reactor 进行通信事件监控 。子 Reactor 线程负责监控各自描述符的读写事件，并进行数据的读写以及业务处理 。
One Thread One Loop：其核心思想是把所有的操作都放到一个线程中进行，一个线程严格对应一个事件处理的循环 。
